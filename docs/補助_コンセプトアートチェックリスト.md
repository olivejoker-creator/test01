# 補助_コンセプトアートチェックリスト

> **役割**：コンセプトアートの「対象一覧＋進捗管理」。工程2でClaudeが依頼文（1種1ファイル）と一緒に作成する。
> 工程3でChatGPTに渡して全体像を把握させる兼、どこまで描いたかを管理する。
> 対象が多いMOD（複数のアイテム/Mob等）ほど有効。

- MOD名：（00から）
- 絵柄スタイル：A:バニラ準拠 / B:バニラ＋ディテール / C:リアル寄り（01で決めたもの・原則A）
- 対象の総数：（例：3）
- 保管場所：`concept/`（依頼文1種1ファイル＋この表）

---

## 3. 運用ルール（厳守）

1. **1対象＝1ファイル**で依頼文を作る（`concept/<アセット名>_依頼.md`）。画像も `concept/<アセット名>.png`
2. ChatGPTでの画像作成は**1対象ずつ**（複数を1枚にまとめない＝混ざり防止）
3. 1つ完成・確認したら、この表の該当行を更新する
4. 描き直し・追加のときは、保管した依頼文を再利用する
