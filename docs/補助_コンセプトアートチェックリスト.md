## 3. 運用ルール（厳守）

1. **1対象＝1ファイル**で依頼文を作る（`concept/<ID>_依頼.md`）
2. ChatGPTでの画像作成は**1対象ずつ**（複数を1枚にまとめない＝混ざり防止）
3. 1つ完成・確認したら、この表の該当行を更新する
4. 描き直し・追加のときは、保管した依頼文を再利用する
