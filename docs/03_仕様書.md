## 1. 概要

プレイヤーは、野生の恐竜（初期はティラノサウルス）を探索し、討伐や巣の探索を通じて卵を入手する。卵を設置して孵化させると、孵化させたプレイヤーの所有恐竜となる。所有恐竜は段階的に成長し、成体になるとサドルを付けて騎乗・収納できる。ホイッスルで個別／集団に命令を出し、図鑑で恐竜情報を閲覧する。

レベル・経験値・テイムゲージ・個体値・遺伝などの育成系要素は **本バージョンでは実装しない**（`00` のスコープに従う）。

### 体験の流れ（ティラノサウルスの場合）

1. 全バイオームに低確率でスポーンする野生ティラノ（敵対・成体）と遭遇する。
2. 討伐すると低確率で卵がドロップする。または自然生成の「巣」から卵を取り出す。
3. 卵を地面に設置（ブロック化）し、熱源を隣接させて孵化を待つ。
4. 孵化した幼体は設置者の所有になる。生肉を与えて回復・成長促進する。
5. 成体に育ったらサドルを付けて騎乗、インベントリに収納、ホイッスルで命令する。
6. 図鑑で恐竜情報を確認する。

## 4. ブロック

### 4-1. ティラノの卵 `tyrannosaurus_egg`

| 項目 | 内容 |
| --- | --- |
| 種別 | Block（＋BlockItem） |
| 設置 | 地面に設置するとブロック化 |
| 孵化条件 | 隣接（上下左右斜め含む近傍）に熱源があると孵化が進行。無いと停止 |
| 熱源（暫定タグ `dinoparadise:incubator_heat_source`） | 焚き火 / 松明 / 溶岩 / マグマブロック |
| 孵化時間 | 24000 tick（MC約1日／Config化） |
| 孵化挙動 | 完了時に幼体を出現させ、卵ブロックは消滅 |
| 所有継承 | 設置時に設置者UUIDを保存し、孵化幼体に飼い主として継承 |
| 入手 | 巣からの取り出し／討伐ドロップ（クラフト不可） |
| 回収 | 設置済みの卵は破壊で消滅（アイテムに戻さない／暫定） |

### 4-2. ティラノの巣 `tyrannosaurus_nest`

| 項目 | 内容 |
| --- | --- |
| 種別 | Block（コンテナ） |
| 生成 | 自然生成のみ（feature 配置）。全バイオームの地上に低頻度 |
| 内包 | 生成時に卵 1〜2 個を内包（Config化） |
| 操作 | 右クリックでGUI。卵を取り出すと空になる（補充なし／暫定） |
| クラフト | 不可 |

| 回収 | 不可（シルクタッチ等でも回収不可／暫定） |

## 5. エンティティ

### 5-1. ティラノサウルス `tyrannosaurus`

| 項目 | 内容 |
| --- | --- |
| 分類 | 恐竜（新カテゴリ）。所有可能 |
| 野生敵対性 | 敵対（プレイヤーを能動攻撃） |
| 食性 | 肉食 |
| 成長段階 | 幼体 → 若年体 → 成体 |
| 性別 | ♂/♀ を孵化・スポーン時に 50/50 で決定（挙動影響なし） |
| 所有 | 孵化により設置者を飼い主に設定。野生スポーン成体は所有化不可 |

### 5-2. 成長段階別ステータス（暫定・Config化）

| 段階 | HP | 攻撃力 | 当たり判定(幅×高) | 移動速度 | 騎乗 | インベントリ |
| --- | --- | --- | --- | --- | --- | --- |
| 幼体 | 40 | 2 | 0.7 × 1.0 | 0.30 | × | × |
| 若年体 | 100 | 6 | 1.3 × 2.0 | 0.30 | × | × |
| 成体 | 200 | 12 | 2.0 × 3.0 | 0.30 | ○ | ○ |

- 防御力は 0 固定（防御力・移動速度育成は対象外。基礎値のみ設定し育成しない）。
- ノックバック耐性は高め（押し負けない想定）。
- これらは育成で変動しない固定値。

### 5-3. 成長・孵化フロー（暫定・Config化）

| 遷移 | 所要時間 | 進行条件 |
| --- | --- | --- |
| 卵 → 幼体（孵化） | 24000 tick | 熱源隣接時のみ進行 |
| 幼体 → 若年体 | 24000 tick | 時間経過。給餌で残り短縮 |
| 若年体 → 成体 | 48000 tick | 時間経過。給餌で残り短縮 |

- 給餌短縮：幼体・若年体に生肉を与えると残り成長時間を一定割合短縮（バニラ幼体方式）。Config で ON/OFF・短縮率。
- 成体は成長停止。

### 5-4. 食性（犬準拠・Q8）

- 全段階、生肉系を与えると HP 回復。満腹（HP満タン）時は食べない。
- 対象（暫定タグ `dinoparadise:tyrannosaurus_food`）：生の牛肉・豚肉・鶏肉・羊肉・ウサギ肉・腐肉。
- 幼体／若年体への給餌は 5-3 の成長短縮を兼ねる。
- 繁殖（love mode）には入らない（繁殖・遺伝は対象外。犬と異なる点）。

### 5-5. AI命令（所有個体・Q11）

| コマンド | 挙動 |
| --- | --- |
| 追従 | 飼い主に付いて移動 |
| 待機 | その場で待機（犬の sit 相当）、移動しない |
| 防衛 | 飼い主の近辺に留まり、飼い主を攻撃した相手／飼い主が攻撃した相手／接近する敵対Mobに反撃 |
| 中立 | 自分から攻撃しない。被攻撃時のみ反撃 |
| 攻撃的 | 索敵範囲内の敵対Mob＋飼い主の標的を能動攻撃 |

- 「攻撃的」の対象は **敵対Mob限定**（村人・パッシブMobは襲わない／暫定。Q-07）。
- 野生個体（未所有）はコマンドの影響を受けず、敵対AIで行動する。

### 5-6. 性別表示（Q9）

- 性別は内部状態として保持。挙動に影響しない。
- **mobにカーソルを合わせた時、名前と一緒に性別（♂/♀）を表示**する。
- 表示方式（視線時HUD／ネームタグ常時表示）は「9. 質問事項 Q-06」。

### 5-7. スポーン・ドロップ（Q14 / Q2 / Q3）

| 項目 | 内容（暫定・Config化） |
| --- | --- |
| バイオーム | 全バイオーム |
| 時間帯 | 昼夜問わず |
| 出現数 | 単独（min=1 / max=1） |
| 重み | weight = 2 |
| 出現段階 | 成体のみ |
| 敵対性 | 野生は敵対 |
| 卵ドロップ | 討伐時 5%（Config化） |

## 6. GUI

### 6-1. 恐竜インベントリ（成体・Q10）

| 項目 | 内容 |
| --- | --- |
| 開き方 | 成体に対し しゃがみ＋右クリック（空手）。馬と同じ操作系 |
| サドルスロット | × 1（ここにサドルを入れると騎乗可能化） |
| 収納スロット | 15（3×5） |
| 備考 | 幼体・若年体はインベントリ無し |

### 6-2. 騎乗（成体・Q10）

| 項目 | 内容 |
| --- | --- |
| 条件 | 成体かつサドル装着済み |
| 乗る | 空手で右クリック |
| 操作 | WASD移動＋ジャンプ（馬準拠） |
| 騎乗中攻撃 | 左クリックで前方噛みつき（暫定。実装難なら後タスクへ分割／Q-05） |

### 6-3. ホイッスル操作（Q5 / Q12）

| 操作 | 効果 |
| --- | --- |
| しゃがみ＋右クリック | 選択コマンドを巡回（追従→待機→防衛→中立→攻撃的）。選択中をアクションバー表示 |
| 恐竜を直接右クリック | その個体に選択中コマンドを適用（個別命令） |
| 空に向けて右クリック | 所有する全恐竜に選択中コマンドを適用（集団命令） |

### 6-4. 図鑑GUI（Q12）

| 項目 | 内容 |
| --- | --- |
| 開き方 | `dino_encyclopedia` を右クリック |
| 構成 | 1ページ1種、左右で種切替 |
| ロック | 無し（最初から全種閲覧可） |
| 表示項目 | 名前／食性／生息地／役割／サイズ／成長段階／解説 |
| 設計 | 将来50種をデータ駆動で追加可能な構造 |
