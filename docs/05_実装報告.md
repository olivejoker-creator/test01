## 3. 仕様との差分・実装上の判断

### 3-1. 仕様どおりに実装した項目

- 成長段階：BABY → JUVENILE → ADULT（時間経過 + 給餌短縮）
- 孵化条件：隣接熱源タグ（`dinoparadise:incubator_heat_source`）
- 所有継承：設置者 UUID → 孵化時に `setTame(true)` + `setOwnerUUID()`
- 性別：孵化/スポーン時に 50/50、表示のみ（挙動への影響なし）
- AI 命令：FOLLOW / SIT / DEFEND / NEUTRAL / ATTACK
- 騎乗：ADULT + サドル装着時のみ（`hasSaddle()` ガード）
- 巣：クラフト不可・設置不可・回収不可（Q-03）
- 卵補充なし（Q-02）
- 野生スポーン：成体のみ（`MobSpawnEvent.FinalizeSpawn` でフォース ADULT）
- 図鑑：クラフト入手・ロックなし（Q-01）
- 騎乗中攻撃：Config で ON/OFF（Q-05、初期値 ON）

### 3-2. 仕様から外れた点・暫定対応

| 項目 | 仕様 | 実装 | 理由 |
| --- | --- | --- | --- |
| バイオームスポーン登録 | Config から weight/min/max 反映 | BiomeModifier JSON はハードコード値（2/1/1）。実行時に Config 値は反映されない | Forge 1.20.1 の BiomeModifier は JSON のため Config 参照不可。デフォルト値が仕様と一致しているため実害なし |
| テクスチャ・モデル | 本実装 | プレースホルダー PNG | アート素材は別途制作が必要 |
| ティラノ以外の恐竜 | 将来50種 | ティラノサウルス 1 種のみ | 初期実装スコープ |
| GUI テクスチャ | encyclopedia.png | 単色プレースホルダー | アート未製作 |
| 騎乗中攻撃（Q-05） | 左クリックで噛みつき | Config フラグのみ実装（攻撃処理未実装） | 仕様書で「後タスク分割可能」とされており初期実装から除外 |
