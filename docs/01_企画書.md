## 📌 このファイルの更新方法（ChatGPTへの依頼ルール）

> 企画や仕様にChatGPTを使うときは、このファイル（と必要なら00）を渡して、次のルールで更新してもらう。
> **毎回この指示を打たなくても、このファイルを渡せば伝わる。**

ChatGPTへ渡すときの依頼（このまま伝えればよい）：

```
このMOD企画書（00基盤定義も渡す場合はそれも）を、以下のルールで更新してください。
1. 既存の見出し・構成（テンプレ形式）は変えず、内容だけ埋める／更新する
2. 00と01の内容に食い違いがあれば整合させる
3. 変更したら、末尾の「更新履歴」に1行追記する（日付・版・変更点）
4. 完成版を .md ファイル形式で返す（無理なら本文テキストで全文）
```

更新の流れ：**最新の00・01を渡す → ChatGPTが更新 → ファイルで受け取り → D&Dで上書き保存 → push**
（ファイルで返らなければ本文をコピペして📥取り込み。2台運用なので渡す前に `git pull` で最新化）
